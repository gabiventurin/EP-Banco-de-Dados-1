Sobre a interface:

Essa interface foi prepara para corresponder aos requisitos do artefato F do EP de Banco de Dados 1 do curso de Sistemas de Informação no segundo semestre de 2024.

Assim, essa interface possui os recursos de criar, alterar e deletar elementos das tabelas xyz do banco de dados desenvolvido previamente na tarefa. Esses recursos podem ser acessados pela barra de navegação no topo da página, onde C simboliza "Create", U simboliza "Update" e D simboliza "Delete".
